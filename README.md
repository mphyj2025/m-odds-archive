# m-odds-archive

## 项目简介

`m-odds-archive` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本项目不针对任何单一域名或具体网站，仅作为一个通用型静态页面存档与展示空间。

所有内容以 HTML 文件的形式存放，可直接通过浏览器打开查看。

## 目录说明

```
m-odds-archive/
├── index.html        # 主页 / 导航页
├── pages/            # 存放归档的独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
├── assets/           # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md         # 本文件
```

- `pages/`：主要归档目录，每个 HTML 文件代表一个独立页面。
- `assets/`：存放页面共用的样式、脚本和图片资源。
- `index.html`：可选的导航页面，用于快速浏览归档内容。

## 页面归档说明

- 所有归档页面均为独立 HTML 文件，不依赖外部数据库或后端服务。
- 页面内容可能包含静态文本、样式、脚本或嵌入资源。
- 文件名采用 `page-xxx.html` 的命名方式，便于识别和排序。
- 如需新增归档页面，请将 HTML 文件放入 `pages/` 目录，并适当更新 `index.html`（如有）。
